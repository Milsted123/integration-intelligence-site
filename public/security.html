Confidentiality & Data

How we handle your documents

Deal documents are sensitive. This page sets out clearly how your materials are handled — who can access them, how AI processing works, and what happens to them after the engagement.

Integration Intelligence operates as a single-operator practice. There is no team, no shared platform, and no automated pipeline processing your documents without review. What follows describes the full picture.

01

Documents stay with the engagement lead

Deal materials you share are received and held by the named engagement lead — not uploaded to a shared platform, a central database, or any third-party service beyond what is disclosed below.

Each engagement is treated as a discrete, independent unit. No content, signals, or findings from one deal are carried into another.

02

Named lead access only

Your documents are accessible only to the named engagement lead identified in your NDA. There is no wider team access, no offshore processing resource, and no automated pipeline with unreviewed access to your materials.

03

AI processing — Anthropic Claude API

All AI processing uses the Anthropic Claude API — not the Claude.ai consumer interface. This distinction matters: API access is configured with zero data retention, meaning prompts and completions are not stored on Anthropic's infrastructure after each processing call completes.

Anthropic does not train on API inputs under zero-retention configuration. Your deal content is processed and then gone — it does not persist on Anthropic's servers.

The Anthropic API is the only point at which document content leaves the engagement lead's direct control. It is the sole sub-processor involved in the service.

04

Report delivery

Analysis reports are delivered directly to the named client contact via a secure link. They are not shared with any other party.

05

Documentation available on request

Sub-processor documentation — including Anthropic's Data Processing Addendum available at the API tier — can be provided on request as part of your organisation's vendor review process.

For any questions about data handling, contact jon@integration-intelligence.com.
